Tip-002：设置 VS Code 的显示语言

在前期的使用过程中，其实英文真的不会造成多大影响，因为菜单什么的，基本不用去看，所以显示语言并不需要修改。

如果想要修改，也很简单，只是可能和我们日常设置软件的操作不太一样而已。

首先打开编辑器，然后按下快捷键 F1 或者 Ctrl+Shift+P 打开命令栏，然后输入 language，就是语言的意思，太长了？输入 lang 就够的

找这一项：Configure Display language，就是“配置显示语言”的意思，上下方向键选中它，然后回车。

就可以选择需要的语言了，zh-cn 就是中文。如果选项里没有，就选择最后一项，安装其他语言（可能是英文的，反正最后一项啦。

这时候编辑器的左侧会像下图这样，列出所有可供安装的语言，我们当然选择 Chinese (Simplified) 就是简体中文啦。点击后面绿色的安装按钮，它会自动安装。

安装完毕之后，把编辑器关掉，重新打开。这时候就可以按这上面步骤选择中文啦。
