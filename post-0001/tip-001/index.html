Tip-001：Visual Studio Code

Visual Studio Code 是微软开发的一款跨平台编辑器，自带了许多实用的功能，开箱即用，支持代码提示和补全，天生前端友好。非常适合作为前端入门的第一款编辑器。

请注意一定在官方网址（https://code.visualstudio.com/）下载，这样才能获得最新的稳定版本，而且无捆绑，不流氓，一路下一步，安心的无脑安装即可。
