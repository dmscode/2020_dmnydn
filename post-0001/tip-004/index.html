Tip-004：代码提示和自动完成

一、提示和补全

一个感叹号，一个 Tab，就变成了一大堆代码，让我们不由得发出深深的赞叹。

首先，这是编辑器知道我们使用了什么编程语言以后，打的优秀辅助。（参见：Tip-003：编程语言是个啥）

大家不都说自己英文不好，不喜欢背单词，记不住，太复杂……等等借口么。现在全不存在啦。你只要打两个字母，编辑器就开始提示，也就是说你记得单词大概长什么样子就行，然后看着提示选择。

当然不只有单词，复杂的结构也有的，你选择到对应的提示上，按一下 Tab，编辑器就自己把相应的内容写出来了。这叫自动补全/自动完成。反正，一个优秀的编辑器，你给点提示，它就自己写代码了。

二、Emmet

HTML 和 CSS 里面一个重要代码补全方案是 Emmet，旧名（Zen Coding）。这些就是作为了解，不用记啦。

这个方案十分简便，强大，经常打几个字就能出来一大堆代码，超爽的说。有兴趣的可以学习一下它的语法，不过这应该是在学习完 CSS 之后的事情。虽然它很简单，但是没有相应的基础，还是没法理解呀。

在 VS Code 里面，自带了 Emmet 方案，就很省心，很舒适。不过，它是通过自身的代码补全功能来顺便实现的，所以在一些细节上，和标准的 Emmet 有些出入。

作为新人，这些不重要，爽就完了~
